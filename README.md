This repository archives historical data from the real-time database.    

The real-time database (https://github.com/StorywithLove/workflow) only retains data of the latest one year, and all historical data beyond this period will be archived to this repository.   

**Sync Rule**   
The automatic synchronization task runs at 09:00 on the 1st day of each month, which synchronizes all data of the previous month from the real-time database to this local archive repository.
